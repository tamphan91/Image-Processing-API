# Image-Processing-API
Image Processing API

## SETUP PROJECT
- npm install
- cp .env.example .env

## SCRIPTS
- npm run prettier
- npm run lint
- npm run test
- npm run dev
- npm run start

## ENDPOINT
- http://localhost:8000/api/images?fileName=fjord
- http://localhost:8000/api/images?fileName=fjord&width=300&height=200
